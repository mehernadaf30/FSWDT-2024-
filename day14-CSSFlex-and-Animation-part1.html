DISPLAY:FLEX is used in parent so that the child gets displayed in flex layout either row or column

FLEX DIRECTION PROPERTIES:
flex-direction:row=It will set the content in row wise.
flex-direction:column=It will set the content in column-wise.
flex-direction:column-reverse.It will reverse the column.
flex-direction:row-reverse=It will reverse the row.

GAP PROPERTY:We can use Gap Property instead of margin also.It will provide gap between boxes syntax:gap:10px;

FLEX- WRAP:The flex-wrap css property sets whether felx items are forced onto one line or can wrap into multiple lines.The flex-wrap property is a quick way to make parent elemenets more responsive on various screen sizes. syntax:flex wrap:wrap;

FLEX-RATIO:We can use this instead of width. Syntax:flex:2; //here the 2 is size which we are giving.

JUSTIFY-CONTENT PROPERTIES: justify-content:flex-start:It will display boxes/content from start.
justify-content:flex-end:It will display boxes/content from end.
justify-content:center:It will display boxes/content in center.
justify-content: space-around:This option will evenly distribute flex items in line with equal space around them.Space around creates equal space around items.
justify-content: space-between:Space between places equal space between items.
justify-content:space-evenly:The space evenly distributes the space betweeen items
evenly.It is similiar to space-around but provides equal instead of half-sized space on the edges.

ALIGN-ITEMS PROPERTIES: For Column:flex-start,flex-end,center,stretch
align-items:flex-start;
align-items:flex-end;
align-items:flex-center;
align-items:flex-stretch;
